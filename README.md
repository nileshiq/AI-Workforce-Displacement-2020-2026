# DisplaceIQ

## Dataset

**File:** `ai_workforce_displacement_global_2020_2026.csv`

**20,800 records** across 100+ countries, 10 industry sectors, and 25 quarters (2020 Q1 – 2026 Q2).

Data is research-calibrated synthetic, grounded in WEF Future of Jobs 2025, Goldman Sachs GenAI Labour Report 2025, McKinsey State of AI 2025, OECD Employment Outlook 2025, and IMF WEO 2025.

### Coverage

| Field | Detail |
|---|---|
| Countries | 100+ with ISO3 codes |
| Regions | North America, Europe, Asia, Africa, South Asia, Latin America, Middle East |
| Income Groups | High Income, Upper Middle, Lower Middle, Low Income |
| Sectors | Technology, Finance, Healthcare, Manufacturing, Retail, Education, Transport, Media, Administrative, Energy |
| Period | 2020 – 2026 (quarterly) |

### Key Columns

| Column | Description |
|---|---|
| `sector_automation_risk_score` | Automation risk per sector [0–1] |
| `ai_adoption_index` | AI adoption level per country [0–1] |
| `pct_sector_workforce_displaced` | % workforce displaced by AI |
| `pct_sector_workforce_new_roles_created` | % workforce in newly created roles |
| `net_workforce_change_pct` | Net change (new roles minus displaced) |
| `ai_cited_layoff_announcements` | Layoff announcements citing AI |
| `ai_skill_wage_premium_pct` | Wage premium for AI-skilled workers |
| `pct_displaced_roles_female` | % of displaced roles held by women |
| `reskilling_programs_count` | Active reskilling programmes |
| `govt_ai_policy_score_1_to_10` | Government AI policy strength |
